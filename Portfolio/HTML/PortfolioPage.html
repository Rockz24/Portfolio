In this project I incorporated JavaScript into my HTML and CSS design to create a simple social media post blog using the Dynamic Object Model manipulation and dynamic webpages.

Using JavaScript, I created a function called addPost() designed to add a new post each time a user interacts with the "add" button. I added the function in my HTML page with the attribute onclick to observe changes within my Document Object Model (DOM).


The HTML code "going to the beach today" is placed within a div element. I had to generate a fresh div element whenever a user adds a post, even if the user does not intend to include any text. To achieve this, I structured my code so that a post is created even if the input is empty. I employed conditionals to implement this, which is illustrated in the following example below.


I also had to display the input provided by the user. This involved generating new paragraphs each time the user clicks the "add" button.

I further developed the functionality by adding like and comment buttons, all achieved using if statements.

I aimed to ensure that when a new post is added, the input box is cleared as well. To accomplish this, I used an ID to set the value of the input box to an empty string ("").

I incorporated a counter to display the number of posts whenever a user adds new content. To achieve this, I introduced a new variable named "counter" and set it to 1, located above the addPost() function. The reason for setting the counter to 1 is due to the pre-existing post ("going to the beach today") present on the webpage.

I specifically intended for the counter to increment only when a user enters text and creates a post. To fulfill this requirement. I implemented an increment operation within the "if" block of the addPost() function.

Subsequently, I introduced a new function called "remove" to facilitate the removal of the alert when the remove notification button is clicked.
